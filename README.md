# udev
fork udev from [systemd 214](https://github.com/vitalikp/systemd)

# Install
 - download [udev](https://github.com/vitalikp/udev/archive/master.tar.gz);
 - unpack archive and go into directory;
 - create **build** directory;
 - run commands:
```
	# cd build
	# cmake ..
	# make
```
